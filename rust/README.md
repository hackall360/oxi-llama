# Oxi Rust

This directory hosts the Rust crates that power Oxi LLaMA. The crates are now part of the root Cargo workspace, so builds and runs should be issued from the repository root.

## Build

From the repository root run:

```bash
cargo build
```

## Run

To execute the CLI entry point from the repository root:

```bash
cargo run
```

The Inno Setup script for packaging is located at `../installer/setup.iss`.
