# Ollama API Examples

The Rust API crate contains runnable examples under `rust/api/examples`.
Run an example from the repository root with:

```shell
cargo run --package api --example <example>
```

Available examples include:

- `chat`
- `generate`
- `generate_streaming`
- `multimodal`
- `openai_chat`
- `openai_completion`
- `pull_progress`
